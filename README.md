# BetterLights

*A Dynamic lighting mod for Garry's Mod.* 

Everything can be configured in the tool menu tab labeled "Better Lights". 

> [!IMPORTANT]
> I Highly recommend that you use the 64-Bit branch for Garry's Mod. It's more stable.
